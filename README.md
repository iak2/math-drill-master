# math-drill-master
Simple  Math Drills for Elementary and Middle School Sctudents

Can be viewed directly from github using the URL:

https://iak2.github.io/math-drill-master/medium-drill.html

## Run locally without localhost URL

Use the PowerShell runner:

```powershell
.\run-html.ps1
```

It starts a static server on `0.0.0.0:8080` and prints a LAN/computer-name URL (for example `http://192.168.1.20:8080/medium-drill.html`) so you do not need to open a `localhost` URL.
